## 一句话说明

MIRAGEA AI 是一个“AI 短剧共创 + 剧情预测市场 + IP 币行情”的互动内容平台。B.AI 负责剧情生成、创作者工具和 IP 币估值解释；HTX 生态负责支付、资产展示、交易、激励和创作者分润。

## HTX 融合点

- 支付方式：剧情投票、预测市场、剧集权益和 IP 币相关交易中展示 HTX 支付入口。
- 资产口径：我的页面资产总览统一以 HTX 展示，方便讲清用户资产、预测资产和 IP 币资产折算。
- 交易与账本：市场动态、成交记录、结算消息和创作者分润都可以进入 HTX 账本体系。
- 激励模型：HTX 可用于高权重投票、手续费折扣、创作者分润、会员权益和用户贡献奖励。

## B.AI 融合点

- 创作者工具：在“我的”页面使用 B.AI 生成剧情分支和内容建议。
- 剧情共创：根据用户投票、评论、点赞和剧情设定辅助生成后续方向。
- IP 币估值：根据预告片、剧情介绍、互动热度、预测市场和交易数据生成估值区间。
- 估值解释：页面展示热度、剧情潜力、风险评分和预测周期，例如“未来 24 小时”。

## 系统闭环

```text
用户看短剧
→ 投票决定剧情
→ 预测市场形成剧情判断
→ B.AI 生成分支和估值解释
→ 每部剧形成 IP 币行情
→ 用户用法币 / U / HTX / IP 币参与支付和交易
→ 创作者获得分润和数据反馈
→ 下一集继续生成
```

## 变现逻辑

- 内容消费：会员、提前看、剧集解锁、番外内容。
- 交易手续费：剧情预测市场和 IP 币相关交易。
- 创作者工具：B.AI 分支生成、封面、预告、标题和数据分析。
- IP 服务：IP 币行情、估值、数据服务和运营工具。
- 品牌合作：剧情分支冠名、IP 联名、互动任务和广告植入。

## 路演建议

演示顺序建议：

1. 发现页：展示短剧观看、点赞评论、剧情投票。
2. 市场页：展示剧情预测、支付方式、成交记录。
3. 剧详情：展示 IP 币页签、预测市场页签和 B.AI 预测周期。
4. 我的页：展示 HTX 资产总览、绑定钱包、创作者工具。

讲解时注意：

- 当前是 Demo 和测试网模拟资产。
- B.AI 估值是内容和市场信号分析，不构成投资建议。
- Demo 阶段可用 GLM 跑真实 AI 生成，后续可替换为 B.AI Key。
